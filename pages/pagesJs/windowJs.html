LOCAL STORAGE

localStorage é uma funcionalidade do JavaScript que permite armazenar dados no navegador do usuário de forma persistente, ou seja, os dados armazenados no localStorage não têm data de expiração, assim permanecendo disponíveis mesmo após o fechamento do navegador ou de páginas web por se manter justamente, ou em um sistemas de arquivos do dispositivo ou em um banco de dados SQLITE, não chega a ser visivel, porém é acessado por meio das ferramentas de desenvolvimento do navegador em questão.

As vantagens do localStorage se definem em manter as informações salvas apenas no mesmo dominio que as criou, mantendo assim um grau de segurança maior. Tem uma capacidade de armazenamento de até 10Mb dependendo do navegador utilizado para salvar todo tipo de informação necessária. Além de contar com comandos simples que podem fácilmente serem implementados assim que necessário assim como os listados abaixo!

  • SET ITEM

    • O comando setItem(key, value) é utilizado para armazenar uma chave e seu respectivo valor na memória do navegador, ambos os valores passados deverão ser string, caso contrário a conversão será feita automaticamente!

    Exemplo :

    SET ITEM

  • GET ITEM

    • O comando getsItem(key) é utilizado para recuperar um valor associado a chave fornecida, caso ela não exista, retornará null

    Exemplo :

    GET ITEM

  • REMOVE ITEM

    • O comando removeItem(key) é utilizado para remover a chave fornecida, caso ela não exista, nada irá acontecer

    Exemplo :

    REMOVE ITEM

  • CLEAR

    • O comando clear() é utilizado para remover todos os itens da memória para o domínio atual

    Exemplo :

    CLEAR

  • LENGTH

    • O comando .length é utilizado para retornar o número de itens armazenados na memória

    Exemplo :

    LENGTH

  • KEY

    • O comando key(index) é utilizado para retornar a chave do item na posição especificada na memória, caso seja inválido retornará null

    Exemplo :

    KEY
